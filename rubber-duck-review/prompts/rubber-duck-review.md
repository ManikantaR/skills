# rubber-duck-review

Portable pointer for non-Claude harnesses (Codex, Copilot, etc.). Full text at
`skills/rubber-duck-review/SKILL.md` in this plugin.

Run before treating any plan, spec, implementation, or fix as done. State each of
the following explicitly, out loud, not just implicitly assumed:

1. The exact problem.
2. The smallest solving change.
3. The invariant that must remain true.
4. The validation that proves success.
5. The next likely failure mode.

If any answer is unclear or hand-wavy, keep refining before calling it ready.
